# CIAO v0.2 Foundation Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在空目录中搭建 CIAO v0.2 的完整工程骨架，覆盖 monorepo、web、shared、engine、providers、worker、Prisma schema 与核心页面/API 入口。

**Architecture:** 采用 `Turborepo + Next.js 14 App Router` 的 monorepo 结构。`apps/web` 提供 CAO-facing surface 和 API routes，`packages/shared` 承载共享类型与常量，`packages/engine` 承载 loop/governor/capabilities 骨架，`packages/providers` 提供 mock 与 provider adapter 抽象，`workers` 预留 BullMQ worker 入口。

**Tech Stack:** TypeScript, Next.js 14, React 18, Tailwind CSS, Prisma, Zustand, Zod, BullMQ, Redis
